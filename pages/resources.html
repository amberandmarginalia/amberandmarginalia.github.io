Webmastery Resources

  • Here's an easy explanations of borders by Solaria, and one from CSS Tricks.
  • W3schools is a good place to learn the basics of HTML and CSS.
  • CSS Tricks has a great guide on flexbox. But if you want to learn with a cute game, try Flexbox Froggy.
  • Mozilla Web Docs has a great explanation on the table element (and many other topics).
  • To learn more about the personal web and for amazing resources and tutorials, check out 32 bit cafe, Melonland and Sadgrl.online.
  • Looking for a free and open source painting software? I use Krita. There's a lot of resources on their website or youtube, and on the community, where you can find free brushes, palettes and plugins. I particularly love rakurri's brushes.
  • CSS Diner is a little game that helps you learn CSS selectors.
